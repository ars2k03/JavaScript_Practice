# JavaScript Practice

A hands-on JavaScript practice repository for learning JavaScript step by step — from beginner concepts to advanced topics.

---

## About

This repository contains multiple JavaScript practice files and examples created for improving JavaScript skills through practical coding.

The goal of this repository is to build a strong understanding of JavaScript fundamentals, module systems, functional programming methods, and asynchronous programming.

## Requirements

Install Node.js before running the files.

Check installation:

```bash
node -v
```

---

## How to Run

### Clone Repository

```bash
git clone https://github.com/ars2k03/JavaScript_Practice.git
```

### Go to Project Folder

```bash
cd JavaScript_Practice
```

### Run JavaScript Files

Example:

```bash
node Basics/variables.js
```

### Run ES Module Files

```bash
node ESM/file_1.mjs
```
